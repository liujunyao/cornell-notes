## 🚀 如何使用

### 方法1：直接打开（推荐）

```bash
# 在浏览器中打开任意页面
open design/prototypes/html-prototype/pages/index.html

# 或者在VSCode中安装 Live Server 扩展，右键选择 "Open with Live Server"
```

### 方法2：本地服务器

```bash
cd design/prototypes/html-prototype

# Python 3
python -m http.server 8080

# 然后访问 http://localhost:8080/pages/index.html
```

### 方法3：集成到项目

将 `html-prototype/` 目录复制到项目根目录：

```bash
cp -r design/prototypes/html-prototype frontend/public/prototypes
```

## 💡 设计亮点

1. **完全遵循设计规范**
   - 所有颜色、字号、间距都使用 CSS 变量
   - 精确匹配设计稿中的尺寸和布局

2. **交互原型**
   - 不是静态页面，包含基本交互逻辑
   - 自动保存、工具栏激活等真实功能

3. **响应式设计**
   - 移动端、平板、PC 三端适配
   - 使用媒体查询和弹性布局

4. **可复用性**
   - CSS 设计系统可直接用于前端开发
   - HTML 结构清晰，便于理解和修改

5. **原型即文档**
   - 代码即设计规范的展示
   - 前端开发可参考 HTML 结构

## 🛠️ 技术栈

- **纯 HTML5 + CSS3**：无框架依赖
- **原生 JavaScript**：基本交互
- **CSS 变量**：设计系统
- **Flexbox + Grid**：布局
- **媒体查询**：响应式
