## ✨ 设计亮点

### 1. 视觉冲击力
- **大面积渐变 Hero 区域**：立即吸引用户注意
- **醒目的白色文字**：在蓝色背景上形成强烈对比
- **清晰的行动号召**：两个按钮提供明确选择

### 2. 信息层次清晰
- **Hero 区域**：品牌和核心价值主张
- **功能区域**：具体功能介绍
- **统计区域**：社会证明和可信度

### 3. 现代化设计风格
- **大圆角卡片** (12px)：柔和、友好
- **渐变背景**：现代、有活力
- **大字号标题**：清晰、易读
- **充足的留白**：不拥挤，易浏览

### 4. 一致的视觉语言
- **统一的间距系统**：16px、24px、32px
- **统一的圆角**：4px (按钮)、12px (卡片)
- **统一的颜色**：蓝色系贯穿始终

## 📊 技术实现

### 渐变实现

```javascript
fill: {
  type: "gradient",
  gradientType: "linear",
  rotation: 135,
  colors: [
    {color: "$primary", position: 0},
    {color: "$primary-light", position: 1}
  ]
}
```

### 透明背景按钮

```javascript
{
  type: "frame",
  fill: "transparent",
  stroke: {thickness: 2, fill: "$white"},
  cornerRadius: "$radius-sm"
}
```

### 居中对齐的统计项

```javascript
{
  type: "frame",
  layout: "vertical",
  alignItems: "center",
  gap: "$spacing-sm"
}
```
