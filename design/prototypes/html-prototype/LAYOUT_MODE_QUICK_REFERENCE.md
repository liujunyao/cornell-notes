# 🎨 智能布局系统 - 快速参考

**版本**: v2.5 | **状态**: ✅ 已实现

---

## ⚡ 核心功能

**自动根据屏幕宽度切换布局模式**

---

## 📊 布局模式速查

| 模式 | 触发条件 | 布局结构 | 适用设备 |
|-----|---------|---------|---------|
| **经典模式** | < 1600px | 线索(25%) + 笔记(75%) + 总结(底部) | 笔记本、普通显示器 |
| **宽屏模式** | ≥ 1600px | 线索(20%) + 笔记(55%) + 总结(25%右侧) | 桌面显示器、宽屏 |

---

## 🔍 如何测试

### 方法 1: 调整浏览器窗口
1. 打开 http://localhost:8080/pages/note-editor.html
2. 拖动窗口边缘调整宽度
3. 跨越 1600px 时观察布局变化

### 方法 2: 开发者工具
1. 按 F12 打开开发者工具
2. 点击"Toggle device toolbar" (Ctrl+Shift+M)
3. 设置不同宽度测试:
   - 1366px → 经典模式
   - 1920px → 宽屏模式

### 方法 3: 查看控制台
打开控制台 (F12 → Console) 查看实时布局信息:
```
═══════════════════════════════════════
🎨 康奈尔笔记编辑器 - 智能布局系统
═══════════════════════════════════════
📐 屏幕宽度: 1920px
🖥️ 布局模式: 宽屏模式（三栏）
   线索栏: 20% | 笔记栏: 55% | 总结栏: 25%（右侧）
═══════════════════════════════════════
```

## 🛠️ 技术细节

### 断点定义
```css
@media (min-width: 1600px) {
  /* 宽屏模式样式 */
}
```

### 检测函数
```javascript
function detectLayoutMode() {
  const screenWidth = window.innerWidth;
  return screenWidth >= 1600;
}
```
