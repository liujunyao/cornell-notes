# 康奈尔笔记预览面板 - 快速参考指南

## 📌 一句话总结
将笔记列表页面右侧预览面板从**统计卡片**改为**康奈尔笔记三要素结构展示**（线索栏 25% | 笔记栏 75% | 总结栏 100%）

---

## 🎯 核心改变

| 方面 | 旧设计 | 新设计 |
|-----|------|------|
| **内容** | 统计卡片（无关） | 康奈尔笔记三要素 |
| **布局** | 网格式卡片 | 三区域分层 |
| **意义** | 冗余显示 | 内容相关 |

## 📂 技术实现

### HTML 类名
```html
.cornell-preview              <!-- 主容器 -->
.cornell-preview-top          <!-- 上半部分（线索+笔记）-->

.cornell-notes-section        <!-- 笔记栏 -->

.cornell-section-label        <!-- 区域标签 -->
.cornell-section-content      <!-- 区域内容 -->
.cue-item                     <!-- 线索栏项目 -->
```

### CSS 关键属性
```css
.cornell-preview-top {
  display: flex;
  gap: var(--spacing-lg);
  height: 400px;
}

.cornell-cue-section {
  flex: 0 0 25%;              /* 固定 25% 宽度 */
}

.cornell-notes-section {
  flex: 1;                    /* 占用剩余空间 */
  overflow-y: auto;           /* 独立滚动 */
}

.cornell-summary-section {
  min-height: 120px;          /* 最小高度 */
}
```

## 📞 常见问题

### Q: 为什么改用三要素结构？
A: 预览面板应该展示选中笔记的实际内容结构，而不是冗余的统计数据。这样用户能直观看到康奈尔笔记法是如何应用的。

### Q: 线索栏为什么是 25% 宽度？
A: 这是康奈尔笔记法的标准比例，经过实践证明最合适的视觉平衡。

### Q: 如何动态更新内容？
A: Phase 2 中会通过 JavaScript 绑定实际笔记数据，在用户选择笔记时自动更新预览。

### Q: 支持深色模式吗？
A: 目前使用浅色主题。深色模式支持计划在 Phase 4 实现。
