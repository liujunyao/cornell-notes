# 配色方案优化 - 2026-01-29

## 问题诊断

### 原配色方案存在的问题：
1. ❌ **背景色对比度不足**：`#F5F5F5` 灰色背景与 `#FFFFFF` 白色面板区别不明显
2. ❌ **视觉感受不佳**：灰色背景让人感觉"脏"、不清爽
3. ❌ **主色调缺乏活力**：`#3A6EA5` 偏灰蓝色，不够明亮
4. ❌ **层次感不清晰**：整体色彩层次不够分明

## 参考设计分析

基于 `design/prototypes/images/` 中的学达优应用：

- **特点**：干净、清爽、专业

## 新配色方案

### 设计理念
**"纯白主义 + 明亮蓝调"**：
- 采用纯白色作为主内容区背景
- 极浅蓝灰色作为页面和侧边栏背景
- 明亮的蓝色作为主色调
- 提升对比度和视觉清晰度

## 视觉效果提升

### 1. 背景层次
```
旧方案：
┌─────────────────┐
│ 灰色背景 #F5F5F5 │  ← 与白色面板对比度不足
│  ┌───────────┐  │
│  │ 白色面板  │  │  ← 不够清晰
│  └───────────┘  │
└─────────────────┘

新方案：
┌─────────────────┐
│ 极浅蓝灰 #F8F9FB │  ← 清爽、有层次
│  ┌───────────┐  │
│  │ 纯白面板  │  │  ← 对比度高，清晰
│  └───────────┘  │
└─────────────────┘
```

### 2. 三栏笔记布局
```
┌──────────────────────────────────────┐
│        标题栏（纯白）                  │
├───────────┬─────────────────────────┤
│ 线索栏    │  笔记栏（纯白）          │
│ #F0F2F5   │                         │
│（浅灰蓝） │  内容区域清晰可读        │
│           │                         │
├───────────┴─────────────────────────┤
│ 总结栏（#F0F2F5 浅灰蓝）             │
└──────────────────────────────────────┘
```

### 3. 按钮交互
```
普通状态：#4361EE（明亮蓝）
悬停状态：#3451C7（深蓝） + shadow-hover
按下状态：#3451C7 + shadow-sm
```

## 设计原则总结

1. **纯白主义**：主内容区使用纯白色（#FFFFFF），保证阅读舒适度
2. **层次分明**：背景（#F8F9FB）、侧边栏（#F0F2F5）、内容（#FFFFFF）三层分明
3. **明亮活力**：主色调从灰蓝变为明亮蓝，提升视觉能量
4. **柔和阴影**：使用双层阴影和带主色调的悬停阴影，增强层次感
5. **高对比度**：确保所有文本与背景的对比度符合 WCAG AA 标准
