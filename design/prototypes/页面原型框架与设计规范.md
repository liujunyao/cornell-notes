# 康奈尔笔记应用 - 页面原型框架与设计规范

本文档详细描述每个核心页面的布局、组件、交互标注，可直接用于Figma设计落地。

## 一、设计系统基础

### 1.1 色彩系统

**主色板**
| 颜色名称 | HEX值 | RGB值 | 用途 | 说明 |
|---------|------|-------|------|------|
| 学术蓝 | #3A6EA5 | 58, 110, 165 | 主按钮、导航激活、重点文字 | 体现专业学习氛围 |
| 知识蓝 | #3498DB | 52, 152, 219 | 核心知识点高亮 | 定义/概念标记 |
| 强调橙 | #E67E22 | 230, 126, 34 | 重点问题高亮、警告 | 重点问题/待解决 |
| 警示黄 | #F1C40F | 241, 196, 15 | 疑问/易混淆高亮 | 需要特别注意 |
| 成功绿 | #27AE60 | 39, 174, 96 | 反思/心得高亮、完成状态 | 个人思考/已掌握 |
| 引用紫 | #9B59B6 | 155, 89, 182 | 引用/来源高亮 | 引文/参考资料 |

**中性色板**
| 颜色名称 | HEX值 | 用途 |
|---------|------|------|
| 深灰#1F2937 | #1F2937 | 主标题、重点文字 |
| 中灰#4B5563 | #4B5563 | 正文、次级标题 |
| 浅灰#9CA3AF | #9CA3AF | 辅助文字、提示 |

**高亮系统（五类内容属性）**
```
1. 核心知识点 → 黄色背景 (#FFFF00) + 黄色索引条
2. 重点问题    → 橙色背景 (#FF9900) + 橙色索引条
3. 关键案例    → 蓝色背景 (#00FF99) + 蓝色索引条
4. 反思心得    → 紫色背景 (#9900FF) + 紫色索引条
5. 警示易错    → 红色背景 (#FF3333) + 红色索引条
```

### 1.2 字体系统

**字体族**
```
优先级：
  1. 中文：思源黑体/苹方/微软雅黑
  2. 英文：-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
  3. 代码：'Monaco', 'Menlo', monospace
```

**字号与层级**
| 层级 | 字号(移动端) | 字号(平板/PC) | 加粗 | 用途 | 行高 |
|-----|-------------|---------------|------|------|------|
| H1-大标题 | 20px | 24px | 是 | 页面标题、笔记标题 | 1.4 |
| H2-中标题 | 16px | 18px | 是 | 分区标题（线索/笔记/总结） | 1.5 |
| 正文 | 14px | 16px | 否 | 笔记内容、按钮文字 | 1.6 |
| 辅助 | 12px | 14px | 否 | 时间、提示、批注 | 1.5 |
| 代码 | 12px | 14px | 否 | 代码块内容 | 1.8 |

### 1.3 间距系统

```
基准单位：8px

间距规范：
  - 极小：4px（组件内间距）
  - 小：8px（相邻元素）
  - 中：16px（区块间距）
  - 大：24px（主要分区）
  - 极大：32px（页面边距）
```

### 1.4 圆角规范

```
- 极小：2px（细线、细节）
- 小：4px（输入框、按钮）
- 中：8px（卡片、弹窗）
- 大：12px（大型容器）
- 圆形：50%（头像、图标按钮）
```

## 二、核心页面框架

### 2.1 首页（Home）- P0

**页面信息架构**
```
┌─────────────────────────────┐
│  状态栏（系统默认）          │ 高度：44px(iOS)/24px(Android)
├─────────────────────────────┤
│  导航栏                      │ 高度：56px
│  [搜索框]         [+新建]    │ 搜索框占80%宽，+按钮40×40px
├─────────────────────────────┤
│  快捷入口（卡片式）         │ 高度：100px
│  ┌─────┐ ┌─────┐ ┌─────┐   │ 3个卡片，横向排列
│  │最近 │ │待复 │ │收藏 │   │
│  └─────┘ └─────┘ └─────┘   │
├─────────────────────────────┤
│  分类&笔记列表              │ 自适应高度
│  [分类1]                    │
│  ├─ 笔记标题 1     2024-01-29│ 卡片高度：60px(移动端)
│  ├─ 笔记标题 2     2024-01-28│
│  └─ 笔记标题 3     2024-01-27│
│                             │
│  [分类2]                    │
│  ├─ 笔记标题 4     2024-01-26│
│  └─ 笔记标题 5     2024-01-25│
│                             │
│  [+新增分类]                │
├─────────────────────────────┤
│  导航栏（底部）              │ 高度：56px
│  [首页] [复习] [协作] [我的]  │
└─────────────────────────────┘
```

**组件详解**

1. **搜索框**
   - 尺寸：宽度占屏80%，高度44px
   - 样式：左侧放大镜图标、右侧清除按钮
   - 交互：输入时显示历史搜索 + 实时联想结果
   - 颜色：背景#F3F4F6，边框#E5E7EB，聚焦时边框变#3A6EA5

2. **快捷新建按钮**
   - 尺寸：40×40px（移动端）、48×48px（平板/PC）
   - 样式：圆形，背景#3A6EA5，图标"+"白色
   - 交互：点击 → 进入三分栏编辑页（空白笔记）
   - 状态：默认、hover、按下

3. **快捷入口卡片**
   - 尺寸：移动端（宽度100%）、平板/PC（宽度30%左右）
   - 样式：圆角8px，白色背景，轻微阴影
   - 内容：
     * 图标（24px）+ 文字标签 + 数字徽章（显示未读或待做数量）
     * 最近编辑：显示最近3-5个笔记
     * 待复习：显示当日待复习笔记数量（基于艾宾浩斯）
     * 收藏笔记：显示星标笔记总数
   - 交互：点击卡片 → 进入对应列表页面

4. **笔记分类列表**
   - 分类标题：字号16px、加粗、颜色#1F2937
   - 笔记卡片：
     * 高度60px（移动端）、72px（平板/PC）
     * 布局：标题（H3，14-16px）+ 编辑时间（12px，灰色）+ 标签/状态（可选）
     * 右侧显示：最后编辑时间、是否未读提示
     * 交互：
       - 单击：进入笔记编辑页
       - 长按/右键：弹出菜单（删除/分享/标星/修改标签）

5. **导航栏（底部）**
   - 尺寸：宽度100%、高度56px（移动端）、64px（平板/PC）
   - 项目：4个（首页/复习/协作/我的）
   - 样式：图标24px + 文字标签（12-14px）
   - 激活状态：图标&文字变主色#3A6EA5，背景略有阴影
   - 非激活状态：图标&文字灰色#9CA3AF

**多端适配**

| 设备 | 尺寸 | 调整 |
|------|------|------|
| 移动端(竖) | 375×812px | 快捷入口纵向排列、列表单列 |
| 平板(竖) | 768×1024px | 快捷入口横向排列、列表可双列 |
| 平板(横) | 1024×768px | 快捷入口3列、列表网格视图 |
| PC(1920) | 1920×1080px | 快捷入口4列、列表网格、右侧显示日历 |

### 2.2 笔记编辑页（Note Editor）- P0（核心）

**页面信息架构（三分栏核心布局）**

```
┌─────────────────────────────────────────────────────────┐
│  状态栏                                                   │ 44px
├─────────────────────────────────────────────────────────┤
│  标题栏                                                   │ 56px
│  [←返回] [笔记标题输入框(居中)] [AI] [分享] [...]       │
├──────────────────────┬──────────────────────────────────┤
│                      │                                   │
│   线索栏             │         笔记栏                    │ 自适应
│   (25%)              │         (75%)                     │
│                      │                                   │
│  ┌──────────────┐   │  ┌──────────────────────────────┐│
│  │ 关键词/问题  │   │  │                              ││
│  │ AI提炼 [◉]  │   │  │  富文本编辑区                ││
│  │              │   │  │  [工具栏:悬浮显示]           ││
│  │ 高亮索引     │   │  │  - 加粗/斜体/列表           ││
│  │ ┼ 黄色      │   │  │  - 高亮/插入图片/公式       ││
│  │ ┼ 橙色      │   │  │  - AI功能                    ││
│  │ ┼ 蓝色      │   │  │                              ││
│  └──────────────┘   │  └──────────────────────────────┘│
├──────────────────────┴──────────────────────────────────┤
│  总结栏(跨左右两栏)                          已保存 14:32  │ 20%
│  ┌─────────────────────────────────────────────────────┐│
│  │  核心提炼                 待解决问题                 ││
│  │  [输入核心收获]          [输入待解决问题]           ││
│  │  AI生成 [◉]              AI生成 [◉]                ││
│  └─────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────┘
```

**组件详解**

1. **标题栏**
   - 返回按钮：左上48×48px，图标24px
   - 笔记标题输入框：
     * 宽度自适应（约60%屏宽）
     * 字号20px(移动端)/24px(PC)、加粗
     * 聚焦时显示光标、失焦时自动保存
     * 默认文字："未命名笔记+时间戳"
   - 右侧按钮区：
     * AI功能（切换面板）、分享、更多菜单
     * 每个按钮48×48px

2. **线索栏（左侧，25%）**
   - 标题："线索" 字号16px、加粗
   - 内容区：
     * 纯文本输入框，支持多行
     * 占位提示：输入关键词、问题等
   - AI提炼按钮：圆角4px、背景#3A6EA5、白色文字
   - 高亮索引条（右侧纵向）：
     * 对应笔记栏高亮内容的位置
     * 宽度8px、高度按内容比例
     * 颜色与高亮类型一致
     * 点击索引条快速跳转到对应笔记内容

3. **笔记栏（右侧，75%）**
   - 标题："笔记" 字号16px、加粗
   - 富文本编辑区（无限滚动）
   - 工具栏（悬浮式）：
     * 鼠标悬停显示、点击隐藏
     * 位置：选中内容上方，固定距离
     * 包含：
       - 基础格式：加粗、斜体、下划线、删除线
       - 列表：有序/无序列表、任务清单
       - 插入：图片、视频、音频、代码块、表格、公式
       - 高亮：5类预设高亮（图标+颜色）
       - AI：自动高亮、AI拓展、AI排版
     * 尺寸：每个按钮36×36px
   - 高亮管理：
     * 选中内容 → 点击高亮按钮 → 选择颜色
     * 右侧自动显示高亮索引条

4. **总结栏（底部，20%高度）**
   - 标题："总结" 字号16px、加粗
   - 两列布局：
     * 左列：核心提炼
     * 右列：待解决问题
   - 每列含AI生成按钮
   - 右上角显示保存状态："已保存 HH:MM"

### 2.3 复习页面（Review Page）- P0/P1

本页面分为三个模式，用户可切换。

**页面信息架构**

```
┌─────────────────────────────────────────────────────────┐
│  状态栏                                                   │ 44px
├─────────────────────────────────────────────────────────┤
│  标题栏                                                   │ 56px
│  [←返回] [笔记标题] [基础] [进阶] [高阶]  [退出]      │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  根据选定模式显示不同内容                               │ 自适应
│                                                          │
├─────────────────────────────────────────────────────────┤
│  底部操作栏                                             │ 56px
│  [上一行] [下一行] [★薄弱点] [完成复习]               │
└─────────────────────────────────────────────────────────┘
```

**模式详解**

#### 模式1：基础模式（遮盖式背诵）

```
中部显示（自适应高度）：
┌─────────────────────┐
│ 线索栏（正常显示）   │
│ 问题1                │
│ 问题2                │
│ ...                  │
└─────────────────────┘

┌─────────────────────────────────────┐
│ 笔记栏（有遮盖层）   [解锁一行▼]    │
│ ┌─────────────────────────────────┐ │
│ │ ███████████████ (遮盖层)        │ │ 第1行
│ ├─────────────────────────────────┤ │
│ │ ███████████████ (遮盖层)        │ │ 第2行
│ ├─────────────────────────────────┤ │
│ │ ███████████████ (遮盖层)        │ │ 第3行
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘

交互：点击任意行 → 该行显示内容，遮盖层消失
```

#### 模式2：进阶模式（AI提问式）

```
中部显示（自适应高度）：
┌─────────────────────┐
│ 线索栏（正常显示）   │
│ 问题1                │
│ 问题2                │
│ ...                  │
└─────────────────────┘

┌─────────────────────────────────────┐
│ AI问题框                            │
│ 请回答：什么是康奈尔笔记法的5R？   │
│ ┌─────────────────────────────────┐ │
│ │ [用户输入答案的文本框]          │ │
│ │                                 │ │
│ └─────────────────────────────────┘ │
│                          [比对▼]    │
├─────────────────────────────────────┤
│ 标准答案：                          │
│ 康奈尔笔记法的5R分别是...           │ （初始隐藏）
│                                     │ （比对后显示）
│ 您的答案正确率：85%                 │
└─────────────────────────────────────┘

交互：
1. 用户在输入框输入答案
2. 点击[比对]按钮
3. 系统对比用户答案与标准答案
4. 显示正确率、标准答案、错误提示
```

#### 模式3：高阶模式（识别式考验）

```
中部显示（自适应高度）：
┌─────────────────────────────────────┐
│ 笔记栏（纯文本，无高亮颜色）       │
│                                     │
│ 康奈尔笔记法是一种系统化的        │
│ 笔记方法，由康奈尔大学...         │ （灰色正文）
│                                     │
│ 核心是5R流程：记录、简化、       │
│ 背诵、反思、复习。                 │
│ ...                                 │
│                                     │
│ 用户自主标记区域：                 │
│ [已标记] 显示用户标记的高亮区域    │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ AI标记对比                          │
│ 您标记的准确率：78%                 │
│ ┌─────────────────────────────────┐ │
│ │ ✓ 标记正确的部分                │ │
│ │ ✗ 遗漏标记的部分                │ │
│ │ ! 错误标记的部分                │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘

交互：
1. 用户读纯文本，自主标记重要部分（鼠标选中即可）
2. 点击[比对]按钮
3. 系统对比用户标记与AI标记
4. 显示准确率与差异提示
```

**共通要素**

- **模式切换按钮**（标题栏）：基础/进阶/高阶，单选，宽度约120px每个，圆角4px
- **底部操作栏**：固定高度56px，4个按钮
  * [上一行]：跳转到笔记栏上一行
  * [下一行]：跳转到笔记栏下一行
  * [★薄弱点]：为当前行标记星标（变为实心五角星，橙色）
  * [完成复习]：弹出复习报告

**完成复习弹窗**

```
┌─────────────────────────────────┐
│ 复习报告                         │ 标题字号20px、加粗
├─────────────────────────────────┤
│                                  │
│ ✓ 复习完成！                    │
│                                  │
│ 复习时长：15分钟 8秒             │ 显示实际复习时长

│ 薄弱点：2个                      │ 标记星标的行数
│                                  │
│ 薄弱点详情：                     │
│ • 康奈尔笔记法的定义             │
│ • 5R流程的具体应用               │
│                                  │
```
