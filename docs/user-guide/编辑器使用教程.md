## 编辑器界面介绍

康奈尔笔记编辑器采用经典的三栏布局，帮助你更有效地记录和组织学习内容。

```
┌─────────────────────────────────────────────────────────────┐
│  ← 返回    [笔记标题]           状态  辅助栏  AI探索  保存  │
├──────────┬──────────────────────────────────────────────────┤
│          │  [工具栏]                                        │
│  线索栏  │  ┌────────────────────────────────────────────┐ │
│  (20%)   │  │                                            │ │
│          │  │         笔记内容编辑区                      │ │
│          │  │                                            │ │
│          │  │                                            │ │
│          │  └────────────────────────────────────────────┘ │
├──────────┴──────────────────────────────────────────────────┤
│  总结栏 (底部)                                              │
└─────────────────────────────────────────────────────────────┘
```

### 顶部操作栏

- **返回按钮**：返回笔记列表
- **标题输入框**：输入笔记标题（默认"未命名笔记"）
- **保存状态**：显示自动保存状态和时间
- **辅助栏按钮**：显示/隐藏左侧线索栏和底部总结栏（专注模式）
- **AI探索按钮**：打开右侧 AI 深度探索面板
- **保存按钮**：手动保存笔记

## 康奈尔三栏结构

### 📌 线索栏（左侧 20%）

**作用**：记录关键词、问题、重点

**使用方法**：
- 在笔记栏记录时，同步在这里写下关键词
- 记录引导性问题，帮助后续复习
- 标注重要概念和术语

**示例**：
```
什么是康奈尔笔记法？
五个步骤
主要优势
适用场景
```

### 📝 笔记栏（中间 80%）

**作用**：详细记录学习内容

**特点**：
- 富文本编辑器，支持多种格式
- 完整的工具栏支持
- 支持插入图片、表格、代码块等

### 💡 总结栏（底部）

**作用**：用自己的话总结整页笔记

**使用建议**：
- 课后或阅读后立即填写
- 用 2-3 句话概括核心内容
- 帮助加深理解和记忆

**示例**：
```
康奈尔笔记法通过三栏结构（线索、笔记、总结）和五步流程（记录、
简化、背诵、思考、复习），系统化地提升学习效率。特别适合课堂
笔记和读书笔记。
```

## 工具栏功能详解

笔记栏顶部的工具栏提供了丰富的编辑功能。

### 1️⃣ 基础格式

| 按钮 | 功能 | 快捷键 | 说明 |
|------|------|--------|------|
| **B** | 加粗 | `Ctrl+B` / `Cmd+B` | 强调重要内容 |
| *I* | 斜体 | `Ctrl+I` / `Cmd+I` | 标注术语或引用 |
| <u>U</u> | 下划线 | `Ctrl+U` / `Cmd+U` | 突出显示 |
| ~~S~~ | 删除线 | - | 标记已完成或废弃内容 |

**使用技巧**：
- 先选中文字，再点击格式按钮
- 或者点击按钮后直接输入格式化文本
- 再次点击按钮取消格式

---

### 2️⃣ 标题

| 按钮 | 功能 | 效果 |
|------|------|------|
| H1 | 一级标题 | # 最大的标题 |
| H2 | 二级标题 | ## 次级标题 |
| H3 | 三级标题 | ### 三级标题 |

**使用场景**：
- **H1**：章节标题
- **H2**：小节标题
- **H3**：要点标题

**示例结构**：
```
# 第一章 操作系统概述
## 1.1 什么是操作系统
### 定义
### 功能
## 1.2 操作系统的发展
```

### 3️⃣ 列表

| 按钮 | 功能 | 说明 |
|------|------|------|
| 1. | 有序列表 | 自动编号的列表 |
| • | 无序列表 | 项目符号列表 |
| ☐ | 任务清单 | 可勾选的待办事项 |

**有序列表示例**：
```
1. 第一步：记录笔记
2. 第二步：提炼线索
3. 第三步：编写总结
```

**无序列表示例**：
```

```

**任务清单示例**：
```
☐ 复习第一章内容

```

### 4️⃣ 插入功能

| 按钮 | 功能 | 说明 |
|------|------|------|
| 🖼 | 插入图片 | 支持图片 URL |
| </> | 插入代码块 | 语法高亮的代码 |
| ❝ | 插入引用 | 引用文字或重要观点 |
| ⊞ | 插入表格 | 创建 3x3 表格 |

#### 🖼 插入图片

1. 点击图片按钮
2. 在弹窗中输入图片 URL
3. 点击确定插入

**支持的图片格式**：
- 在线图片：`https://example.com/image.jpg`
- Base64 编码图片

#### </> 插入代码块

适合记录编程代码、命令行等。

**示例**：
```python
def hello_world():
    print("Hello, Cornell Notes!")
```

#### ❝ 插入引用

适合记录书籍摘抄、名言警句等。

**示例**：
> "系统化笔记，提升学习效率" —— Walter Pauk, 康奈尔大学

#### ⊞ 插入表格

自动创建 3 行 3 列的表格，可手动添加行列。

**示例**：
| 特性 | 优点 | 缺点 |
|------|------|------|
| 三栏结构 | 逻辑清晰 | 需要时间整理 |
| 问题导向 | 强化记忆 | - |

### 5️⃣ 智能高亮

康奈尔笔记编辑器提供 **4 种高亮颜色**，帮助你分类标注重要内容。

| 按钮 | 颜色 | 用途 | 快捷键 |
|------|------|------|--------|
| ⚫ | 黑色 | 还原为普通文本 | - |
| 🟠 | 橙色 (#FF9900) | 重点问题 | `Ctrl+1` |
| 🟢 | 绿色 (#00FF99) | 关键案例 | `Ctrl+2` |
| 🔴 | 红色 (#FF3333) | 警示易错 | `Ctrl+3` |

**使用方法**：
1. 选中需要高亮的文字
2. 点击对应颜色按钮
3. 文字会被**加粗**并**设置前景色**

**效果示例**：
- **橙色高亮**：什么是操作系统的核心功能？
- **绿色高亮**：Linux 内核调度算法实现
- **红色高亮**：注意：死锁的四个必要条件缺一不可

**颜色使用建议**：

| 颜色 | 适用内容 | 示例 |
|------|---------|------|
| 🟠 橙色 | 核心问题、考点、重点概念 | "什么是多态？"、"三大范式" |
| 🟢 绿色 | 实例代码、案例分析、实践经验 | "单例模式实现"、"HTTP 请求示例" |
| 🔴 红色 | 易错点、警告、注意事项 | "注意空指针"、"易混淆概念" |
| ⚫ 黑色 | 取消高亮，恢复普通文本 | - |

**组合使用示例**：

```
操作系统的四个核心功能：

1. 🟠 进程管理 - 什么是进程调度算法？
   🟢 示例：Linux CFS 完全公平调度器
   🔴 注意：进程和线程的区别容易混淆

2. 🟠 内存管理 - 虚拟内存的作用？
   🟢 示例：分页机制的实现
   🔴 警告：内存泄漏的常见原因
```

### 6️⃣ AI 辅助功能

| 按钮 | 功能 | 说明 |
|------|------|------|
| ✨ | AI 自动高亮 | （开发中）自动识别重点并高亮 |
| 💡 | AI 提炼线索和问题 | 自动从笔记中提炼关键线索 |

#### 💡 AI 提炼线索和问题

**功能说明**：
- 自动分析笔记内容
- 提炼 3-8 条关键线索和问题
- 直接插入到左侧线索栏

**使用步骤**：

1. **编写笔记内容**
   - 在笔记栏输入你的学习内容
   - 确保内容足够详细（至少 10 个字符）

2. **保存笔记**
   - 必须先保存笔记（点击右上角"保存"按钮）
   - 或者等待自动保存完成

3. **点击 💡 按钮**
   - 系统状态显示"AI 提炼中..."
   - 等待 3-10 秒

4. **查看结果**
   - 提炼的线索自动添加到左侧线索栏
   - 如果线索栏已有内容，新线索会追加在后面
   - 成功提示："成功提炼 X 条线索"

**示例**：

**笔记内容**：
```
康奈尔笔记法（Cornell Note-taking System）是由康奈尔大学教授
Walter Pauk 在 1950 年代开发的系统化笔记方法。该方法将笔记页面
分为三个区域：右侧的笔记栏用于记录详细内容，左侧的线索栏用于
记录关键词和问题，底部的总结栏用于概括总结。使用流程包括五个
步骤：记录、简化、背诵、思考、复习。
```

**AI 提炼的线索**：
```
什么是康奈尔笔记法？
由谁开发？
三个区域分别是什么？
五个步骤
线索栏的作用
总结栏的作用
如何使用该方法？
```

**注意事项**：
- ⚠️ 笔记内容太少（少于 10 字符）无法提炼
- ⚠️ 新笔记必须先保存才能使用此功能
- ⚠️ 需要配置 AI 服务才能使用（详见管理员配置）

## 使用技巧

### 📖 高效记录流程

**1. 课堂/阅读时（记录阶段）**
```
1. 在笔记栏快速记录内容
2. 暂时不管格式，专注内容
3. 使用快捷键快速格式化
```

**2. 课后整理（简化阶段）**
```
1. 使用工具栏美化格式
2. 添加标题、列表结构
3. 插入图片、表格补充
4. 使用高亮标注重点
```

**3. 提炼线索（背诵阶段）**
```
1. 点击 💡 AI 提炼线索按钮
2. 检查并补充关键问题
3. 确保线索能覆盖所有要点
```

**4. 编写总结（思考阶段）**
```
1. 在总结栏用自己的话概括
2. 不看笔记栏，凭记忆写
3. 2-3 句话，抓住核心
```

**5. 定期复习（复习阶段）**
```
1. 遮住笔记栏，只看线索栏
2. 尝试回忆详细内容
3. 对照笔记栏检查遗漏
```

## 常见问题

### Q1: 为什么 AI 提炼线索按钮无法使用？

**A**: 可能的原因：
1. 笔记内容太少（至少 10 个字符）
2. 笔记未保存（新建笔记必须先保存）
3. AI 服务未配置（联系管理员）

### Q2: 图片无法插入怎么办？

**A**: 确保：
1. 图片 URL 有效且可访问
2. URL 以 http:// 或 https:// 开头
3. 使用支持的图片格式（jpg, png, gif, webp）

### Q3: 高亮颜色可以自定义吗？

**A**: 当前版本提供 3 种预设颜色（橙、绿、红），暂不支持自定义颜色。

### Q4: 快捷键无效怎么办？

**A**:
1. 确保光标在编辑器内
2. 检查是否与浏览器快捷键冲突
3. 尝试点击工具栏按钮代替

### Q5: 自动保存失败怎么办？

**A**:
1. 检查网络连接
2. 手动点击"保存"按钮
3. 查看控制台错误信息
