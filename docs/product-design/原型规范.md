# 康奈尔笔记应用 - Figma可导入原型标注模板

# 一、模板说明（Figma导入指南）

## 1.1 导入方式

1. 将本文档导出为PDF格式（保留标注与排版）；
2. 打开Figma，新建空白文件→点击左上角「文件」→「导入」→选择导出的PDF；
3. 导入后自动生成对应页面，组件库需手动添加至Figma左侧组件面板（参考1.3组件库规范）；
4. 交互热区需在Figma中通过「原型」面板关联，本文档已标注热区位置与跳转关系。

## 1.2 模板结构

模板分为「组件库」「核心页面原型」「标注图层」三部分，严格遵循Figma设计规范，适配移动端（375×812px）、平板端（768×1024px）、PC端（1920×1080px）三端尺寸。

# 二、组件库（Figma可直接复用）

## 2.1 基础组件（P0，全端通用）

* 按钮组件：
  * 快捷新建按钮：圆形，图标为「+」，关联交互：点击跳转至笔记编辑页；
  * 操作按钮（保存/删除/分享）：矩形，圆角4px，文字居中，删除按钮默认禁用，需选中内容后激活；
  * AI功能按钮：带AI图标，文字标注功能名称（如“AI提炼”），加载状态显示转圈动画（Figma需添加原型动画）。
* 输入框组件：
  * 搜索框：左侧放大镜图标，右侧清除按钮，输入时实时联想（Figma添加文本框交互）；
  * 笔记标题输入框：居中对齐，默认文字“未命名笔记+时间戳”，聚焦后清空默认文字；
  * 问答输入框（进阶复习模式）：左侧关联问题文字，右侧提交按钮，适配移动端输入。
* 提示组件：
  * 弹窗提示：居中显示，半透明背景遮罩，含标题、内容、确认/取消按钮（如删除二次确认）；
  * 悬浮卡片：圆角8px，阴影效果，用于AI知识点拓展、批注回复，点击空白处关闭；
  * 加载提示：转圈图标+文字“加载中...”，居中显示，覆盖对应操作区域。

## 2.2 导航组件（P0，全端通用）

* 主导航栏（底部）：
  * 图标+文字组合，4个选项（首页/复习/协作/我的），激活状态文字+图标变主色；
  * 移动端高度56px，平板/PC端64px，图标尺寸24px，文字字号14px（移动端）/16px（平板/PC）。
* 分栏导航（笔记编辑页）：
  * 线索栏/笔记栏/总结栏标题，悬浮显示工具栏入口，点击标题可快速切换焦点栏；
  * 平板横屏模式下，线索栏标题为抽屉触发按钮，点击展开/收起线索栏。

## 2.3 编辑组件（P0，核心功能）

* 工具栏组件：
  * 悬浮式工具栏：默认隐藏，鼠标悬停/手指长按调出，含基础编辑、格式插入、高亮、AI功能按钮；
  * 高亮选择面板：弹出式，5类高亮选项（图标+颜色+文字），选中后自动关闭面板。
* 分栏组件：
  * 线索栏/笔记栏/总结栏容器：带边框，可拖拽调整宽度（Figma添加拖拽交互热区）；
  * 高亮索引条：右侧纵向排列，对应高亮内容位置，点击索引跳转（Figma添加位置关联）。
* 协作组件（P1）：
  * 在线成员头像：小圆形，尺寸28px，显示在编辑页右上角，重叠头像自动排列；
  * 批注标记：红色小旗帜图标，关联对应内容，点击显示批注文字。

## 4.2 动画效果设置（统一体验）

* 页面跳转：淡入淡出，时长0.3秒；
* 组件展开/收起：滑动动画，时长0.2秒，缓动效果“ease-in-out”；
* 加载状态：旋转动画，时长1秒，循环播放；
* 高亮提示：闪烁动画，时长0.5秒，循环2次，颜色变浅。

# 五、导入后优化建议

1. 组件库整理：导入后将所有组件拖拽至Figma左侧「组件」面板，按“基础组件/导航组件/编辑组件”分类归档，便于快速调用；
2. 样式应用：在Figma「样式」面板创建对应颜色、字体变量，统一应用至所有组件，避免样式混乱；
3. 交互关联：按本文档标注的交互逻辑，在Figma原型面板完成所有组件的交互关联，测试跳转流畅性；
4. 适配调整：针对不同设备尺寸，微调组件位置与大小，确保响应式效果符合标注要求；
5. 版本备份：完成优化后保存为Figma组件库文件，单独备份，便于后续迭代复用。
